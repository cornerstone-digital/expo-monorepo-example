# Mobile App

The main Expo React Native application for the monorepo.

## Running the App

```bash
# Start the development server
npm run start

# Run on iOS
npm run ios

# Run on Android
npm run android

# Run on Web
npm run web
```

## Features

This app demonstrates:
- Using shared UI components from `@expo-monorepo/ui`
- Using feature packages from `@expo-monorepo/features/auth`
- NX monorepo structure with TypeScript path mapping

## Dependencies

- `@expo-monorepo/ui` - Shared UI component library
- `@expo-monorepo/features-auth` - Authentication feature package
